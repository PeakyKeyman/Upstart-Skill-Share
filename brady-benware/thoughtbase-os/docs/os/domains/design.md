# Design Domain Handbook

## Purpose

This handbook orients work that affects layout, interaction design, visual direction, and device-specific presentation.

## Current Stance

- Thoughtbase is opinionated about visual direction and interaction quality.
- Mobile-first does not mean mobile-only; desktop layouts must feel native to desktop.
- Durable design worldview lives in policy files, not inside skills.

## Use This Handbook When

- changing page structure or responsive behavior
- adjusting modals, overlays, or route presentation
- applying or revising visual standards
