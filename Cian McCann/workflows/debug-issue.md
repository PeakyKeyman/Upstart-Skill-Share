# Debug Issue Workflow

1. **Debug** → 4-phase scientific investigation (`/custom:debug`)
2. **Fix** → TDD fix (RED: reproduce, GREEN: fix, REFACTOR: clean)
3. **Review** → Focused review of fix (`/custom:review`)
4. **Verify** → Regression check + broader suite (`/custom:verify`)
5. **Document** → Update KNOWN_ISSUES.md + Cognee `save_interaction`
