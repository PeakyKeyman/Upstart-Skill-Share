# New Project Workflow

1. **Common Ground** → Surface assumptions (`/custom:common-ground`)
2. **Codebase Mapping** → Understand existing code (`/custom:architect`)
3. **Brainstorm + Plan** → Explore approaches, create plan (`/custom:plan`)
4. **Execute** → TDD implementation (`/custom:execute`)
5. **Review** → Two-stage code review (`/custom:review`)
6. **Verify** → Goal-backward verification (`/custom:verify`)

**Tip**: Use CodeGraph MCP for step 2 to quickly map the codebase structure.
